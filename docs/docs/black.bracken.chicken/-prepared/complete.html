fun complete(): R

Submits call while blocking then returns the received value is transformed into R; done in synchronous.

Exceptions

UnknownHostException - if couldn't reach/connect host.

NotFoundException - if couldn't find specified api.

UnsupportedMediaTypeException - if didn't support media type.

ExceedRateLimitException - if exceeded rate limit.

IllegalRequestException - if received response-code is not 200 and unexpected.
